# Airline Reservation System
Airline Reservation System using C, GTK GUI, and MySQL

## Tech Stack:
- Language: C
- GUI: GTK
- Database: MySQL

## Modules:
- Booking Tickets
- Viewing Flights
- Canceling Reservations
- Admin Panel
